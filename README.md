# Гит хуки
- Склонировать репозиторий в корень проекта
- Запустить команду в корне своего проекта
```
docs/init.bash
```

## Игнорирование файлов и папок
файл `hooks/.ignore`
Каждая строчка можем содержать регулярное выражение по которым исключаются файлы например:
```
badFile.*
upload/.*
testFile\.php
```
